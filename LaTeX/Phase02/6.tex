\subsection*{\underline{دیتابیس NoSQL}}

در این بخش باید قسمت های مربوط به CRM را که تعدادی از آن ها را داخل Postgres پیاده سازی کرده بودیم، این بار داخل MongoDB پیاده سازی کنیم.

دستورهای مربوط به ساخت کالکشن های این دیتابیس به این صورت است:

\setLTR

کوئری های مربوط به این بخش در ادامه قرار دارند:




کوئری اول:
\setLTR
\begin{lstlisting}
db.hotel_ticket.find({ type: "cancel" }).pretty();
\end{lstlisting}
\setRTL

کوئری دوم:
\setLTR
\begin{lstlisting}
db.phone_call_trip.aggregate([
{
	$group: {
		_id: "$company_id",
		ticketCount: { $sum: 1 }
	}
},
{
	$sort: { ticketCount: -1 }
}
]);
\end{lstlisting}
